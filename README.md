# UnityFlagr

unity client for flagr

[![Build Status](https://travis-ci.org/5minlab/unityflagr.svg?branch=master)](https://travis-ci.org/5minlab/unityflagr)

## features

* PostEvaluation
* PostEvaluationBatch

## build

```sh
cd UnityFlagr
dotnet bulid
```

## test / usage

`UnityFlagrSample/Program.cs`

```sh
cd UnityFlagrSample
dotnet run
```
